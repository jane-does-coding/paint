## Setup

### 1. Clone the repo

```bash
git clone https://github.com/jane-does-coding/paint
cd crumbledpaper
```

### 2. Install the dependencies

```bash
npm i
```

### 3. Configure .env

```bash
DATABASE_URL="your mongodb"
NEXTAUTH_SECRET="random-secret"
GEMINI_API_KEY="gemini api key"
RESEND_API_KEY="resend api key"
NEXT_PUBLIC_APP_URL="public website url"
```

### 4. Setup Prisma

```bash
npx prisma generate
```

### 4. Run the project

```bash
npm run dev
```
